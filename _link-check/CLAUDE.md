# タスク: 日英サイト間のリンク整合性チェック

## 目的
日本語版サイト（ルート）と英語版サイト（`en/`）の間で、リンク切れ・対応ファイル不足・翻訳漏れがないかを定期的に確認する。

## チェック項目
1. **ファイル対応確認**: 日本語版に存在するHTMLファイルに対応する英語版があるか
2. **リンク先確認**: 各ページ内のリンクが正しいファイルを指しているか（`../posts/` と `posts/` の混在など）
3. **翻訳漏れ確認**: 英語版ページに日本語テキストが残っていないか
4. **言語切替リンク確認**: 日本語ページの「English」リンク・英語ページの「日本語」リンクが正しく機能するか
5. **新規ファイル確認**: 日本語版に新しく追加されたファイルに対応する英語版が作られているか

## 作業フロー
1. Claudeが上記チェックを実施し、問題点をリストアップ
2. 問題があれば修正提案または自動修正を行う
3. チェック結果をこのフォルダ内にMarkdownで記録する
例: `check-report-2026-04.md`

## チェック対象ファイル
- ルート直下の `.html` ファイル
- `posts/` フォルダ内の `.html` ファイル
- `en/` フォルダ内の `.html` ファイル
- `en/posts/` フォルダ内の `.html` ファイル
